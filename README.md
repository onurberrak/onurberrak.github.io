# onurberrak.github.io

Personal portfolio site for Onur Berrak — Lead Key Account Manager, Berlin.

## Stack
Plain HTML + CSS + a single inline `<script>`. No build step.
Hosted on GitHub Pages.

## Local preview
Open `index.html` directly in any modern browser:
```
file:///C:/Users/onurb/portfolio/index.html
```

## Deploy
Push to the `main` branch of this repository. GitHub Pages serves
the root automatically because the repo name is `<user>.github.io`.
